# Portafolio web - Teoría de la Programación

Página web estática para GitHub Pages basada en dos PDF:

- `assets/pdf/unidad1.pdf`: Unidad 1
- `assets/pdf/unidad2.pdf`: Unidad 2

## Cómo subir a GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube todos los archivos de esta carpeta al repositorio.
3. En GitHub, entra a **Settings > Pages**.
4. En **Branch**, selecciona `main` y carpeta `/root`.
5. Guarda los cambios.
6. GitHub generará un enlace público para la página.

El archivo principal es `index.html`.
