Algoritmo

Un algoritmo es un conjunto ordenado, finito y definido de instrucciones que permite solucionar un problema, realizar un cálculo o procesar datos. Recibe datos de entrada, los procesa mediante reglas lógicas y produce una salida.

Ejemplo: cálculo de galones

El algoritmo convierte la producción diaria ingresada en litros a su equivalente en galones y calcula el total a cobrar según el precio por galón.

Algoritmo calculodegalones
  Definir litros, precioPorGalon Como Real
  Definir galones, totalACobrar Como Real

  Escribir "Ingrese la producción del día en litros:"
  Leer litros

  Escribir "Ingrese el precio por galón:"
  Leer precioPorGalon

  galones <- litros / 3.785
  totalACobrar <- galones * precioPorGalon

  Escribir "Equivalente en galones:", galones
  Escribir "Total a cobrar: $", totalACobrar
FinAlgoritmo

Estructuras condicionales

Permiten que un programa tome decisiones según una condición. El programa evalúa si una expresión es verdadera o falsa y ejecuta una acción dependiendo del resultado.

Condicional simple

Si condición Entonces
  Acción
FinSi

Condicional doble

Si condición Entonces
  Acción 1
Sino
  Acción 2
FinSi

Condicional múltiple

Según variable Hacer
  Caso 1: Acción 1
  Caso 2: Acción 2
  De Otro Modo: Acción 3
FinSegún

Estructuras repetitivas

Permiten ejecutar instrucciones varias veces mientras se cumpla una condición o hasta completar una cantidad determinada de repeticiones.

Mientras

  Instrucciones
FinMientras

  Instrucciones

Ejercicio: control de ventas en una tienda

Se desea ingresar la cantidad de productos vendidos. Por cada producto se ingresa su precio. Si el precio es menor o igual a cero, se muestra un error y se vuelve a pedir el dato. Al final, si el total supera $100, se aplica un descuento del 10%.

ElementoDescripción
EntradaCantidad de productos y precio de cada producto.
ProcesoValidar datos, acumular total, verificar descuento y calcular total final.
SalidaTotal de venta, descuento aplicado y total a pagar.

Diagrama de flujo del ejercicio

Inicio
Leer cantidadProductos
¿cantidadProductos > 0?
total = 0; i = 1
¿i ≤ cantidadProductos?
Leer precio
¿precio > 0?
total = total + precio; i = i + 1
¿total > 100?
descuento = total × 0.10 / descuento = 0
totalFinal = total - descuento
Mostrar total, descuento y total final
Fin

Codificación en C

#include <stdio.h>

int main() {
  int cantidadProductos, i;
  float precio, total = 0;
  float descuento, totalFinal;

  do {
    printf("Ingrese la cantidad de productos vendidos: ");
    scanf("%i", &cantidadProductos);
    if (cantidadProductos <= 0) {
      printf("Ingrese una cantidad mayor que 0\n");
    }
  } while (cantidadProductos <= 0);

  for (i = 1; i <= cantidadProductos; i++) {
    do {
      printf("Ingrese el precio del producto #%i: ", i);
      scanf("%f", &precio);
      if (precio <= 0) {
        printf("Ingrese un precio valido\n");
      }
    } while (precio <= 0);
    total = total + precio;
  }

  if (total > 100) {
    descuento = total * 0.10;
  } else {
    descuento = 0;
  }

  totalFinal = total - descuento;

  printf("Total sin descuento: $%.2f\n", total);
  printf("Descuento aplicado: $%.2f\n", descuento);
  printf("Total a pagar: $%.2f\n", totalFinal);

  return 0;
}
